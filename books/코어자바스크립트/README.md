# 📚 코어 자바스크립트

> `2024. 05. 07 ~ 2024. 07. 07`  
> Typescript 개발 경험과 높은 이해가 있으신 분 Node 진영에서 Typescript/Javascript로 백엔드 개발을 하고 있지만 가끔씩 찔린다. 너 Javascript 제대로 알고 있는 거 맞아?
